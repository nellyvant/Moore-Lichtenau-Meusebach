README für LAS zu Rasterdaten

1. How to find it
Gehe ins Geoportal Thüringen, finde den Download für DGM, scrolle nach unten zu
Übersichten zu Airborne Laserscanning, Digitale Geländemodelle & Digitale Oberflächenmodelle, welche zeigen wann Gebiete in Thüringen beflogen wurden,
finden Sie hier.

DGM Daten (in den verschienen Jahrgängen) stehen ebenfalls über den standardisierten Atom Feed Service DGM (im Viewer) zum Download bereit.

DOM Daten (in den verschienen Jahrgängen) stehen ebenfalls über den standardisierten Atom Feed Service DOM (im Viewer) zum Download bereit.

LAS Daten (in den verschienen Jahrgängen) stehen ebenfalls über den standardisierten Atom Feed Service LAS (im Viewer) zum Download bereit.

hinter LAS Daten muss man auf "im Viewer" klicken. Man landet hier:https://geoportal.geoportal-th.de/gaialight-th/_apps/atomfeedexplorer/?#feed=https%3A%2F%2Fgeoportal.geoportal-th.de%2Fdienste%2Fatom_th_hoehendaten_las

2. Download
Öffne den Feed mit Klick auf die Karte oder Liste der Einzeldateien öffnen (das dauert eine Weile)
Aktuelle Laserscanning Daten raussuchen und wähle in der Kartenansicht die gewünschten Messtischblätter aus und lade sie herunter
